/*
题目描述
有一个已排好序的数组，要求输入一个数后，按原来排序的规律将它插入到数组中。

假设数组长度为10，数组中前9个数（这9个数要求从键盘上输入，输入时要满足自小到大的输入顺序）已经按从小到大进行排序。

然后再从键盘上输入一个整数，将此整数插入到前有序的9个数中，使得最终的10个数依然是从小到大有序的。

输入
第一行输入以空格分隔的9个整数数，要求按从小到大的顺序输入。

第二行输入一个整数

输出
从小到大输出这10个数，每个数一行。
样例输入
1 11 21 31 41 51 61 71 81
45
样例输出
1
11
21
31
41
45
51
61
71
81
提示
定义数组时，把数组长度定义为10.
真的要哭死哦，学长说我可能考虑还不全面	T_T
*/
#include<stdio.h>
int main(){
	int a[10];
	int j,x,i;
	int temp;
	int record_i;
	for(i=0;i<9;i++){
		scanf("%d",&a[i]);
	}
	for(i=1;i<=8;i++){
		for(int q=0;q<9-i;q++){
			if(a[q]>a[q+1]){
				temp=a[q];
				a[q]=a[q+1];
				a[q+1]=temp;
			}
		}
	}
  	scanf("%d",&x);  
  	for(i=0;i<9;i++){
  		if(x>a[i]){
  			record_i=i;
  			break;
  		}
  	}
	for(j=9;j>record_i;j--) {
		a[j]=a[j-1];
        	a[j]=x; 
	}

   	for(i=0;i<10;i++)  
     	printf("%4d\n",a[i]);  
 	return 0;
}
